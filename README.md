# Discord RPC Example

## Requirements
1. Node.js
2. Discord.exe (Running)

## Steps
Step 1: Download the Repo. <br>
Step 2: Fill ClientID of a Application in `rpc.js` and Save. <br>
Step 3: Run `install.bat` to install Dependencies. <br>
Step 4: Run `start.bat` to start the RPC. <br>
